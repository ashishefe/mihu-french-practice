# Mihika's French Quest

A Vercel-ready `Next.js` site that converts Mihika's Grade 7 French worksheets and practice paper into a balanced learning game.

## What it includes

- Source-linked learning stages based on the uploaded files
- Worked examples and practice questions
- A spaced-repetition memory loop
- Light gamification with health, lives, streaks, and recovery
- A guided writing quest for `Ma sortie récente`
- Local persistence through `localStorage`

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Import this folder into Vercel.
2. Framework preset: `Next.js`
3. Build command: `npm run build`
4. Output: default Next.js output

No environment variables are required.
